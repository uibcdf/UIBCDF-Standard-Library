uibcdf_stdlib.colors.is_hex¶

uibcdf_stdlib.colors.is_hex(color)[source]¶

Method to check if a color code is hexadecimal (HEX)

This method returns the value True if the input argument corresponds to an hexadecimal color code.

Parameters

color (:obj:) – Color code

Returns

True if the input color code takes the hexadecimal (HEX) form.

Return type

bool

Examples

>>> from uibcdf_stdlib.colors import is_hex
>>> is_hex('#008080')
True
>>> is_hex([0.0, 0.5, 0.5])
False
